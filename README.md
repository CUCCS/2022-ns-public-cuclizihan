### 三、新建一个NAT网络

##### 解决方法

参考CSDN上的资料后，得知新建一个NAT网络的方法。

点击`管理-全局设定-网络`，然后点击`新建一个NAT网络`即可。

### 四、kali无法登录

##### 问题截图

![图片加载失败](24.png)

##### 解决方法

出现引导界面时选择恢复模式advance options for kali GNU/Linux：

![图片加载失败](25.png)

直接按‘e’，就进入编辑模式：

![图片加载失败](26.png)

ro改成rw，在gz后加上init=/bin/bash，之后便可以进入命令行，输入`passwd`便可以重置密码：

![图片加载失败](23.png)
